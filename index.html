What is CSS?

CSS, which stands for "Cascading Style Sheets", is a web development language used to style and position the elements which have been coded onto a web page. In short, CSS governs the appearance of a website's HTML.

CSS Syntax:

A CSS rule consists of a selector and a block of declarations. The declaration block is encased in curly braces. It may be written over single or multiple lines. Multiple lines allow for increased readability if a lot of declarations are being applied to the same selector.

Single-line CSS:

h1 { color: #FFF; font-size: 100px; }

Multi-line CSS:

h1 {
  color: #FFF;
  font-size: 100px;
}

In the above examples, h1 is our selector. Color and font-size are our declarations. Declarations are made up of property-value pairs.

Common Declarations: Text and Font

The following table outlines common property-value pairs which are used for styling text and fonts.

Common selectors for these declarations may be:

  • Heading tags: <h1>, <h2>, <h3>, <h4>, <h5> and <h6>.
  • Paragraphs: <p>.
  • Anchor (link) tags: <a>.
  • User-defined classes and ids.

font-family
  • An ordered list of fonts, separated by a comma.
 Defines the type face which will be used by text within the HTML document. Fonts in the list which have multiple words in their name must be surrounded by an inverted comma ('). MDN
font-style
  • normal;

 Defines whether a font is normal, italic or oblique. MDN

font-weight

  • A number between 1 and 1000 (inclusive).
 Defines how bold a font appears to the user. MDN

  • none;

  • none;

word-spacing
  • normal;
  • An em measurement.

  • A pixel measurement.
  • A percentage.
 Defines the amount of spacing between each word in a block of text. MDN
